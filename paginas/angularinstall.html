Pasos de instalacion de Angular

1. Instalar Angular CLI Globalmente

  • npm install -g @angular/cli


    • 2. Ejecutar los Comandos de la CLI de Angular

  • ng --version


    • 3. Crear un Espacio de Trabajo Inicial para la Aplicación

  • ng new my-app


    • 4. Ejecuta la Aplicación Angular en tu Navegador

  • cd my-app


    • A continuación, inicia el servidor de desarrollo ejecutando el siguiente comando:

  • ng serve
